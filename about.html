About DailyPulse

DailyPulse is a modern blog and news website created to bring clean, easy‑to‑read articles to readers on any device. It focuses on short, clear stories that help you stay updated without feeling overwhelmed.

This project was built as a complete frontend website using semantic HTML, responsive CSS, and JavaScript. It is designed to be fast, mobile‑friendly, and accessible so that users on phones, tablets, and laptops can enjoy the same smooth experience.

The homepage highlights the latest news, while the Articles page groups hand‑picked stories and guides. A dynamic “Latest News” section integrates with a real news API to show fresh headlines directly inside the website.

The site also includes a contact form so visitors can share feedback, suggestions, or collaboration ideas. All forms are validated in the browser to reduce errors and improve the overall experience.

Goals of This Project

  • Practice planning and building a complete multi‑page website.
  • Use responsive design so the layout works on mobile and desktop.
  • Apply accessibility best practices with clear structure and labels.
  • Integrate a third‑party API to display real news data.
  • Prepare a project that can be deployed and shown in a portfolio.

Technologies Used

  • HTML5 for structure and semantic content.
  • CSS3 for styling, layout, and responsive design.
  • JavaScript for interactivity, form validation, and API calls.
  • NewsAPI for fetching live news articles.
